Create a real AI output attestation.

Use this form for DAO recommendations, agent reports, code audit findings, or any AI-generated artifact. The API computes hashes, stores the artifact locally as a 0G adapter, and writes the registry record.

Wallet signature is used when available; otherwise the dev issuer is recorded.

Process

From AI result to public receipt.

(01)

Commit

Hash the input, output, and model metadata. The issuer signs the commitment.

(02)

Store

Persist artifact metadata through the 0G Storage adapter and keep sensitive content hash-only when needed.

(03)

Anchor

Write the attestation ID, issuer, trust level, storage pointer, and timestamp to the registry.

(04)

Verify

Anyone can recompute the hash and check issuer, revocation, and optional TEE evidence.
